# 나의 할 일 관리 앱

## 1. 앱 소개
이 앱은 자신의 할 일을 직접 작성하고 직접 완료할 수 있는 할 일 관리 앱 입니다.

## 2. 사용한 AI
깃허브에 있는 AI를 사용했습니다.

## 3. 처음 입력한 프롬프트
HTML,CSS,JavaScript 를 하나의 index.html 파일에 넣어서 할 일 관리 앱을 만들어줘. 할 일을 입력하고 추가할 수 있고, 완료 표시와 삭제 기능이 있었으면 좋겠어. 초보자도 이해할 수 있게 코드를 너무 복잡하게 만들지 말아줘. 라고 프롬프트 입력했습니다.

## 4. 수정 요청한 내용
처음부터 완벽해서 배경색만 수정해달라고 요청했습니다.

## 5. 어려웠던 점 또는 배운 점
어려웠던 점은 전혀 없었고 AI 하나만으로도 손쉽게 앱 하나를 만들 수 있는 것이 신기했습니다. 약간 여러방면에서 자주 사용햐보고 싶습니다.
